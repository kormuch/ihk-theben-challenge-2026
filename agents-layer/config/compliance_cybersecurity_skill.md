# Cybersecurity Certification Compliance Agent Skill

Scope: CRA readiness, SBOM, vulnerability handling, secure update process, dependency traceability, and software lifecycle evidence.

Inputs:

- Connectivity and firmware attributes.
- SBOM evidence.
- Secure update process evidence.
- Vulnerability handling evidence.
- Product lifecycle state.

Checks:

- Connected products have SBOM and update evidence.
- Security evidence is traceable.
- Restricted vulnerability details are access-controlled.

Output: advisory finding with restricted handling marker where needed.
