# Data Protection & Privacy Compliance Agent Skill

Scope: GDPR, telemetry, service data, consent, retention, deletion, and access logging evidence.

Inputs:

- Telemetry and customer data attributes.
- Privacy assessment evidence.
- Data flow and retention evidence.

Checks:

- Telemetry use is declared.
- Privacy evidence exists for products handling personal data.
- Restricted evidence is access-controlled.

Output: advisory finding requiring human privacy review where relevant.
