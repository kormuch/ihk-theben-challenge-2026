# Wireless Standards Compliance Agent Skill

Scope: Wi-Fi, Bluetooth, NFC, radio module evidence, regional approvals, and product labeling.

Inputs:

- Wireless protocol attributes.
- Radio test reports.
- Module qualification evidence.
- Target market.

Checks:

- Wireless claims have evidence.
- Regional market access evidence is linked.
- Labeling and module evidence are traceable.

Output: advisory finding with missing evidence and market-access review recommendation.
