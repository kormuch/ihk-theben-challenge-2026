# DPP Readiness Expert

Scope: DPP field completeness, Data Matrix identity, public DPP view readiness, B2B and authority view readiness, and missing DPP evidence.

Checks:

- GTIN, batch/lot number, and serial number exist.
- DPP fields are traceable.
- Public URL base is stable.
- Authority-only fields are not exposed to public views.

Output: readiness status, score, missing fields, missing evidence, and product workflow feedback.
